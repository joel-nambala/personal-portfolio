About my coding journey

I started coding back in 2019. The first resources I used were YouTube tutorials. I started up with tutorials from traversy media where I learnt the basic fundamentals of HTML5 and CSS3.

I joined freecodecamp where I learnt intensive HTML5 amd CSS3. I received my first certificate of Responsive web design. I then did some basic projects to ensure I have captured the content very well.

I decided to purchase a course from Udemy where I learnt intensive fundamentals of the JavaScript language. I created some projects using the skills I have gained in JavaScript. Am still building up my skills in JavaScript and adding new knowledge to my existing knowledge base.
